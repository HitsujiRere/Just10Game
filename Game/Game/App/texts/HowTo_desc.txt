上から数字の書いてあるセルを落としていき、"Just10"が起きるとスコアが増えるゲームです
"Just10"とは、長方形で囲んだ範囲の合計が「ちょうど10」になるとセルが消えることです
しかし、1番上の「×」のところにセルを置くとゲームオーバーになります
また、セルを消すと相手にオジャマセルを送ることができます
オジャマセルは、"Just10"によって消えたセルに隣接していると消えます

xキー,ECSキー : 戻る　　F4キー : フルスクリーン　　F5キー : 画面サイズの変更

左プレイヤー
【キャラ】A,Dキー : 選択キャラを変更する　　Sキー : キャラを確定する　　Wキー : -----
【バトル】A,Dキー : 落とすセルを移動させる　　Sキー : セルを落とす　　Wキー : -----

右プレイヤー
【キャラ】左,右キー : 選択キャラを変更する　　下キー : キャラを確定する　　上キー : -----
【バトル】左,右キー : 落とすセルを移動させる　　下キー : セルを落とす　　上キー : セルをホールドチェンジする

※「1人であそぶ」ではどちらのプレイヤーの操作でもできます
